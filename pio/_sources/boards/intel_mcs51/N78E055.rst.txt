.. _board_intel_mcs51_N78E055:

Generic N78E055
===============

.. contents::

Hardware
--------

Platform :ref:`platform_intel_mcs51`: The Intel MCS-51 (commonly termed 8051) is an internally Harvard architecture, complex instruction set computer (CISC) instruction set, single chip microcontroller (uC) series developed by Intel in 1980 for use in embedded systems.

.. list-table::

  * - **Microcontroller**
    - N78E055
  * - **Frequency**
    - 22MHz
  * - **Flash**
    - 16KB
  * - **RAM**
    - 1.25KB
  * - **Vendor**
    - `Nuvoton <https://www.nuvoton.com/products/microcontrollers/8bit-8051-mcus/standard-8051-series/n78e055a/?utm_source=platformio.org&utm_medium=docs>`__


Configuration
-------------

Please use ``N78E055`` ID for :ref:`projectconf_env_board` option in :ref:`projectconf`:

.. code-block:: ini

  [env:N78E055]
  platform = intel_mcs51
  board = N78E055

You can override default Generic N78E055 settings per build environment using
``board_***`` option, where ``***`` is a JSON object path from
board manifest `N78E055.json <https://github.com/platformio/platform-intel_mcs51/blob/master/boards/N78E055.json>`_. For example,
``board_build.mcu``, ``board_build.f_cpu``, etc.

.. code-block:: ini

  [env:N78E055]
  platform = intel_mcs51
  board = N78E055

  ; change microcontroller
  board_build.mcu = n78e055

  ; change MCU frequency
  board_build.f_cpu = 22118400L

Debugging

:ref:`piodebug` currently does not support Generic N78E055 board.
