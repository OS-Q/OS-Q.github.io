Installation
============

.. note::

    You do not need to install |PIOCore| if you are going to use :ref:`pioide`.
    |PIOCore| is built into PIO IDE and you will be able to use it
    within PIO IDE Terminal. See instructions on how to open
    :ref:`PIO Core CLI in PIO IDE for VSCode <ide_vscode_pio_core_cli>`.

    If you need |PIOCore| outside PIO IDE, please :ref:`piocore_install_shell_commands`.
