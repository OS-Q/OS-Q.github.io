Generic ML51TD1AE¶

Contents

  • Generic ML51TD1AE

    • Hardware

    • Configuration

    • Debugging

Hardware¶

Platform Intel MCS-51 (8051): The Intel MCS-51 (commonly termed 8051) is an internally Harvard architecture, complex instruction set computer (CISC) instruction set, single chip microcontroller (uC) series developed by Intel in 1980 for use in embedded systems.

Microcontroller

ML51TD1AE

Frequency

24MHz

Flash

64KB

RAM

4.25KB

Vendor

Nuvoton

Configuration¶

Please use ML51TD1AE ID for board option in “platformio.ini” (Project Configuration File):

[env:ML51TD1AE]
platform = intel_mcs51
board = ML51TD1AE

You can override default Generic ML51TD1AE settings per build environment using board_*** option, where *** is a JSON object path from board manifest ML51TD1AE.json. For example, board_build.mcu, board_build.f_cpu, etc.

[env:ML51TD1AE]
platform = intel_mcs51
board = ML51TD1AE

; change microcontroller
board_build.mcu = ml51td1ae

; change MCU frequency
board_build.f_cpu = 24000000L

Debugging¶

Debugging currently does not support Generic ML51TD1AE board.
