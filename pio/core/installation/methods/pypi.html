Warning

We recommend using this method ONLY FOR Continuous Integration systems or where your have full permissions to install PIO Core into the global scope of your OS.

For personal using, and avoiding maintenance and upgrade issues, we HIGHLY RECOMMEND using Installer Script (Recommended) which installs PIO Core into an isolated virtual environment and does not affect your OS.

The latest stable version of PIO Core may be installed or upgraded via Python Package Manager (pip) as follows:

pip install -U platformio
